# 📚 Book Alchemy - Digital Library Management

A full-stack web application built with **Flask** and **SQLAlchemy** to manage a personal library. This project was developed as part of the MSIT Masterschool programm.

## 🚀 Features
* **Author Management:** Add and delete authors with birth and death dates.
* **Book Inventory:** Add books with ISBN, title, and publication year.
* **Smart Filtering:** Search and filter books by title, author, or year.
* **Dynamic Sorting:** Sort your library by various criteria in ascending or descending order.
* **Responsive Design:** A clean, grid-based UI for a modern reading experience.

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/DEIN-USERNAME/book-alchemy.git](https://github.com/DEIN-USERNAME/book-alchemy.git)
   cd book-alchemy
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application:**
   
   ```bash
   python3 app.py
   ```


## 🗄️ Database
The project uses **SQLite** with SQLAlchemy ORM. The database file is located in the `/data` directory.
